# 제5장: 함수 완전 정복

DuckDB는 분석 워크로드에 최적화된 방대한 내장 함수 라이브러리를 제공한다. 표준 SQL 함수부터 JSON 처리, 통계 분석, 복합 타입 조작까지 한 곳에서 해결할 수 있다. 이 장에서는 실무에서 자주 쓰이는 함수를 카테고리별로 정리하고, 각 함수의 동작 원리와 활용 패턴을 함께 살펴본다.

> **이 장의 구성:** 집계 → 문자열 → 날짜/시간 → JSON → 수학/통계 → 리스트·맵·구조체 → 윈도우 함수 심화 순으로 진행한다. 각 절은 독립적으로 참조할 수 있으므로, 필요한 카테고리부터 먼저 읽어도 무방하다.

```sql
-- FILTER 절을 활용한 조건부 집계
SELECT
    COUNT(*) FILTER (WHERE dept = 'Engineering') AS eng_count,
    AVG(salary) FILTER (WHERE active = true)     AS active_avg,
    SUM(amount) FILTER (WHERE year = 2024)       AS sales_2024
FROM employees;
```

---

## 주요 학습 내용

- `COUNT(*)`와 `COUNT(열)`은 다르다. `COUNT(*)`는 모든 행을 세지만 `COUNT(열)`은 NULL을 제외한다. 의도에 맞게 구분해 사용해야 한다.
- `DATE_TRUNC('month', ts)`는 시계열 집계에서 가장 많이 쓰이는 함수다. 월별·주별·일별 집계를 할 때 거의 빠지지 않는다.
- JSON의 `->` 연산자는 JSON 타입을 반환하고 `->>`는 문자열을 반환한다. 비교 연산에 사용할 때는 `->>`로 문자열로 추출한 뒤 처리하거나 명시적 캐스팅이 필요하다.
