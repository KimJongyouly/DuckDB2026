# 14-4. ODBC / ADBC

### ODBC (Open Database Connectivity)

ODBC는 언어에 무관하게 모든 SQL 도구에서 DuckDB에 연결할 수 있게 한다. Excel, Power BI, Tableau, SAS 같은 BI 도구 연동에 사용한다.

```bash
# ODBC 드라이버 설치 (Linux)
wget https://github.com/duckdb/duckdb/releases/latest/download/duckdb_odbc-linux-amd64.zip
unzip duckdb_odbc-linux-amd64.zip
# libduckdb_odbc.so → /usr/local/lib/

# odbcinst.ini 등록
cat >> /etc/odbcinst.ini << 'EOF'
[DuckDB Driver]
Description=DuckDB ODBC Driver
Driver=/usr/local/lib/libduckdb_odbc.so
EOF

# DSN 등록 (~/.odbc.ini)
cat >> ~/.odbc.ini << 'EOF'
[DuckDB_Analytics]
Driver=DuckDB Driver
Database=/home/user/analytics.duckdb
EOF
```

```bash
# macOS
brew install unixodbc
wget https://github.com/duckdb/duckdb/releases/latest/download/duckdb_odbc-osx-universal.zip
# 설치 후 ODBC 관리자에서 드라이버 등록
```

```python
# Python pyodbc로 ODBC 연결
import pyodbc

# DSN 연결
conn = pyodbc.connect('DSN=DuckDB_Analytics')

# 직접 연결 문자열
conn = pyodbc.connect(
    'Driver={DuckDB Driver};'
    'Database=/path/to/analytics.duckdb;'
)

cursor = conn.cursor()
cursor.execute("SELECT * FROM employees WHERE dept = ?", ('Engineering',))

for row in cursor.fetchall():
    print(row)

conn.close()
```

```r
# R에서 ODBC 연결
library(odbc)

con <- dbConnect(odbc::odbc(),
    dsn      = "DuckDB_Analytics",
    Database = "analytics.duckdb"
)

result <- dbGetQuery(con, "SELECT COUNT(*) FROM sales")
dbDisconnect(con)
```

### Power BI / Tableau 연결

```
Power BI에서 DuckDB 연결
────────────────────────────────────────────────────────
1. DuckDB ODBC 드라이버 설치 (Windows용)
2. Windows ODBC 데이터 원본 관리자에서 DSN 생성
   - 드라이버: DuckDB Driver
   - Database: C:\data\analytics.duckdb
3. Power BI Desktop → 데이터 가져오기
   → ODBC → DuckDB_Analytics DSN 선택
4. 테이블/뷰 목록에서 원하는 것 선택

Tableau에서 DuckDB 연결
────────────────────────────────────────────────────────
1. ODBC 드라이버 설치 후 DSN 생성
2. Tableau Desktop → 연결 → 기타 데이터베이스(ODBC)
3. DSN 선택 → 로그인
4. 스키마·테이블 선택
```

### ADBC (Arrow Database Connectivity)

ADBC는 Arrow를 네이티브 전송 포맷으로 사용하는 차세대 데이터베이스 연결 인터페이스다. ODBC보다 직렬화 오버헤드가 적어 대용량 데이터 전송에 유리하다.

```bash
pip install adbc-driver-duckdb adbc-driver-manager
```

```python
import adbc_driver_duckdb.dbapi as duckdb_adbc
import pandas as pd
import pyarrow as pa

# 연결
conn = duckdb_adbc.connect()                        # 인메모리
conn = duckdb_adbc.connect("analytics.duckdb")      # 파일

# 기본 쿼리
cursor = conn.cursor()
cursor.execute("SELECT * FROM employees WHERE dept = 'Engineering'")

# Arrow Table로 가져오기 (Zero-copy)
arrow_table = cursor.fetch_arrow_table()
print(arrow_table.schema)
print(arrow_table.num_rows)

# Pandas DataFrame으로 변환
df = arrow_table.to_pandas()

# Arrow Table → DuckDB (대량 삽입)
data = pa.table({
    'id':     [10, 11, 12],
    'name':   ['Frank', 'Grace', 'Heidi'],
    'dept':   ['Engineering', 'Sales', 'Marketing'],
    'salary': [91000.0, 75000.0, 69000.0]
})
cursor.adbc_ingest('employees', data, mode='append')

conn.commit()
conn.close()
```

```python
# ADBC + SQLAlchemy 연동
from sqlalchemy import create_engine
import pandas as pd

# ADBC를 백엔드로 SQLAlchemy 엔진 생성
engine = create_engine("adbc+duckdb:///analytics.duckdb")

# pandas read_sql
df = pd.read_sql("SELECT * FROM employees", engine)

# to_sql: DataFrame → DuckDB
new_data = pd.DataFrame({
    'id':     [20, 21],
    'name':   ['Ivan', 'Judy'],
    'dept':   ['Engineering', 'HR'],
    'salary': [88000.0, 65000.0]
})
new_data.to_sql('employees', engine, if_exists='append', index=False)
```

```python
# ADBC 스트리밍 처리 (대용량)
import adbc_driver_duckdb.dbapi as duckdb_adbc

conn = duckdb_adbc.connect("huge.duckdb")
cursor = conn.cursor()

cursor.execute("SELECT * FROM massive_table")

# RecordBatch 단위로 스트리밍
reader = cursor.fetch_record_batch(batch_size=100_000)
total = 0
for batch in reader:
    total += len(batch)
    # 배치 처리...
    print(f"  처리: {total:,}행", end='\r')

print(f"\n완료: {total:,}행")
conn.close()
```
