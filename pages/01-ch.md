# 제1장: 설치와 환경 구성

DuckDB의 가장 큰 장점 중 하나는 설치가 놀랍도록 간단하다는 것이다. 별도의 서버 데몬을 띄울 필요도, 복잡한 설정 파일을 편집할 필요도 없다. 이 장에서는 운영체제와 언어별로 DuckDB를 설치하는 방법을 정리하고, CLI에서 첫 쿼리를 실행해보는 것까지 다룬다. 설치 후에는 단 한 줄의 SQL로 데이터 파일을 바로 쿼리할 수 있다는 것도 직접 확인한다.

```bash
# CLI 설치 (macOS/Linux)
curl https://install.duckdb.org | sh

# Python 설치
pip install duckdb

# 설치 확인
duckdb --version
```

---

## 주요 학습 내용

- DuckDB는 별도 서버 없이 설치 즉시 실행 가능한 임베디드 OLAP 데이터베이스다. 설치는 `pip install duckdb` 또는 CLI 바이너리 다운로드로 끝난다.
- CLI 도트 명령어(`.timer`, `.mode`, `.headers`)와 `~/.duckdbrc` 파일을 활용하면 반복 작업을 자동화하고 출력을 원하는 형식으로 제어할 수 있다.
- DuckDB UI(`duckdb -ui`)는 로컬 브라우저 기반 GUI로, 외부 도구 없이도 테이블 탐색과 쿼리 실행이 가능하다.
